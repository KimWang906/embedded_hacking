• 동작 설정
    • 동작 설정을 OFF 로 하면 무선 기능을 사용할 수 없습니다.
  • 동작 모드
    • AP모드 - 가장 일반적인 모드로 무선 랜카드를 가진 컴퓨터의 접속을 허용하여, 무선 통신이 가능하게 합니다.
    • 랜카드모드 - 탑재된 무선 모듈을 이용하여 다른 AP로 접속합니다. 최초로 인식된 한대의 유선 PC만 무선을 통한 통신이 가능합니다.
    • 무선WAN모드 - 탑재된 무선 모듈을 이용하여 다른 AP로 접속합니다. 또한, 탑재된 무선 모듈은 WAN포트로 이용되며, 기존의 유선 WAN포트는 동작이 중지 됩니다.
    • 랜카드 모드 및 무선WAN모드는 일부 제품에서만 지원됩니다.
  • 네트워크이름(SSID)
    • 무선 네트워크를 구분할 수 있는 식별자 입니다.
  • 모드
    • 모드는 무선랜 카드와의 호환성을 설정합니다. (제품별로 지원할 수 있는 모드가 다를 수 있습니다.)
      • 11a/n 5GHz : 5GHz 대역의 11a/n 기반의 무선 호스트가 접속할 수 있습니다.
      • B,G,N : 802.11n/11b/11g 기반의 무선 호스트가 접속할 수 있습니다.
      • G,N : 802.11g/11n 기반의 무선 호스트가 접속할 수 있습니다.
      • N Only : 802.11n 기반의 무선 호스트만 접속할 수 있습니다.
      • G Only : 802.11g(54Mbps)기반의 무선 호스트만 접속할 수 있습니다.
      • G and B : 802.11g(54Mbps)와 802.11B(11Mbps)기반의 무선 호스트가 접속할 수 있습니다.
      • B Only : 802.11b(11Mbps) 기반의 무선 호스트만 접속할 수 있습니다.
  • 지역
    • 현재 사용하고 있는 지역을 선택합니다.
  • 채널
    • 사용할 채널을 선택합니다.
      • 40MHz 채널을 지원하는 제품에는 제어 채널의 위치에 따라서 상위/하위 로 설정할 수 있습니다.
      • 예를 들면,
        • 6번 하위 채널 선택 -> 제어 채널은 6번이며, 6 ~ 10 의 채널을 사용하게 됩니다.
        • 6번 상위 채널 선택 -> 제어 채널은 6번이며, 2 ~ 6 의 채널을 사용하게 됩니다.
      • "채널 검색" 기능을 이용하여 사용하고 있는 지역에서 최적의 채널을 검색할 수 있습니다.
  • SSID(네트워크이름)알림
    • 이 기능은 SSID(네트워크이름)을 모든 무선 호스트에 알려주는 기능입니다.
    • 이 기능을 끔으로써 무선 네트워크상에서 공유기를 숨길 수 있습니다.
    • 네트워크이름을 알고 있는 컴퓨터만 접속이 가능하게 하려면 이 기능을 끄셔야 합니다.

  • 암호화 방법
    • 보안 암호화 설정을 통하여 허가된 사용자만이 무선랜을 통하여 접속할 수 있습니다.
    • 인증방법 : 무선 사용자가 암호화된 인증을 통하여 접속이 가능하게 할 수 있습니다.
      • 개방모드 - 암호화된 인증을 사용하지 않습니다.
      • 공유모드 - 설정된 Key 값을 사용하여 암호화된 인증을 합니다.
          (WEP64와 WEP128의 암호화 방법을 사용할 수 있습니다.)
      • 자동 - 위의 두가지 경우를 모두 지원합니다.
      • WPAPSK, WPA2PSK - 802.11i에 근거한 보다 향상된 인증방법을 사용합니다.
          (TKIP과 AES의 암호화 방법을 사용할 수 있습니다.)
      • ※ WPAPSK/WPA2PSK 사용시에는 두가지 인증방법으로 모두 접속할 수 있습니다.
    • 암호화 방법 : 무선 통신의 보안을 위하여 데이타를 암호화 합니다.
      • 사용안함: 암호화를 사용하지 않습니다.
      • WEP 64 : 64Bit Key를 사용하여 암호화 합니다.
      • WEP 128 : 128Bit Key를 사용하여 암호화 합니다.
      • TKIP, AES : 8자 이상의 문자열을 입력 합니다.
    • 입력 방법 : 암호화에 사용되는 KEY를 입력하는 방법을 선택합니다.
      • 문자열 : 문자열로 입력합니다.
      • 16진수 : 16진수로 입력합니다.
      • KEY 입력 규칙 :
        • 하나의 KEY는 모든 칸을 채워서 입력해야 합니다.
        • 기본 KEY는 항상 입력해야합니다.
        • 64Bit-문자열 :5개 , 128Bit-문자열 : 13개
        • 64Bit-16진수 :10개 , 128Bit-16진수 : 26개 ( 숫자 및 A-F 영문자만 입력가능 )
