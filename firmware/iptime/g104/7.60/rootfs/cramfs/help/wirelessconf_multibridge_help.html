• 개 요
    • 멀티 브리지 기능을 이용하면, 다른 AP로의 브리지 또는 WAN포트 연결이 가능합니다.
  • 사용 방법
    • 동작 설정을 [실행] 을 선택합니다.
    • 무선 동작 모드를 선택합니다.
    • [ 연결할 AP검색 ]을 클릭하여 연결될 AP를 선택합니다.
    • AP 선택 시, 채널,인증 방법,암호화 방법이 자동으로 설정됩니다.
    • 암호화된 AP로의 접속시에는 네트워크 키를 입력합니다.
    • [적용] 버튼을 누릅니다.
    • [다시보기] 버튼을 이용하여, "무선 연결 상태" 를 확인합니다.
  • 무선 동작 모드
    • 멀티 브리지로 사용
      • 연결된 AP로의 무선 연결을 브리지로 사용합니다.
      • 무선 연결을 통과시 모든 소스 MAC주소는 브리지 맥주소로 변환됩니다.
      • IP프로토콜 이외의 프로토콜은 통신할 수 없습니다.
    • 무선 WAN으로 사용
      • 연결된 AP로의 무선 연결을 WAN포트로 사용합니다.
      • [ 인터넷 연결 설정 ]을 통하여 무선WAN포트를 설정할 수 있습니다.
      • 기존의 유선 WAN포트를 사용할 수 없습니다.
  • 브리지 맥주소
    • 연결될 AP로의 접속 시에 사용되는 무선 MAC주소를 의미합니다.
  • 무선연결상태
    • 연결 상태 및 연결된 AP이 맥주소(BSSID)를 표시합니다.
  • 네트워크 이름
    • 연결될 AP의 이름을 검색 하거나 직접 입력합니다.
  • 채 널
    • 수동 채널 설정
      • 연결될 AP의 무선 통신 채널을 수동으로 설정합니다.
      • 연결되어 있는 AP의 무선 채널이 바뀔 경우, 재 설정을 하여야 합니다.
      • AP기능[무선 설정/보안]에서 사용되는 채널도 변경됩니다.
    • 자동 채널 설정
      • 연결될 AP의 무선 통신 채널을 자동으로 검색하여 설정합니다.
      • 연결되어 있는 AP의 무선 채널이 바뀔 경우, 재 설정할 필요가 없습니다.
      • 연결될 AP와의 연결이 끊어진 경우, 본 장치의 AP기능이 정상적으로 동작하지 않습니다.
  • 인증방법 및 암호화방법
    • 연결될 AP의 인증,암호화,키를 동일하게 입력합니다.
