• 포트포워드 설정
    • 공유기의 외부 네트워크 사용자가 내부 네트워크에 접속하고자 하는 경우 사용되는 어플리케이션의 포트번호를 내부 네트워크의 IP 주소와 내부포트 번호로 연결시켜주는 기능이 필요한데, 이런 기능을 포트포워드라고 합니다.
  • 정의된 리스트
    • 미리 정의된 규칙을 이용하거나, 사용자가 임의로 규칙을 정의 할 수 있습니다.
  • 규칙이름
    • 임의로 입력가능하며 구분이 쉽도록 입력합니다.
  • 내부 PC IP 주소
    • 어플리케이션이 설치된 내부서버 또는 PC의 IP 주소를 입력합니다.
  • 프로토콜
    • Protocol Type (TCP/UDP)을 선택 합니다.
  • 외부 포트
    • 외부 인터넷에서 접속시 사용되는 내부서버 또는 PC에 설치된 어플리케이션 포트범위를 설정합니다.
  • 내부 포트
    • 내부서버 또는 PC에 설치된 어플리케이션 포트범위를 설정합니다.
    • '외부 사용 포트 범위'와 동일한 경우에는 입력하지 않아도 됩니다.
