• 컨넥션 검색의 개념
    • [전체 컨넥션 상태] 검색 - 통신중인 컨넥션에 대한 요약 정보를 표시합니다.
    • [IP별 컨넥션 상태] 검색 - 특정 PC(IP)가 점유하고 있는 인터넷 대역폭(MB/s),사용량(Bytes), 그리고 컨넥션 수를 검색할 수 있습니다.
    • [TCP/UDP 포트별 컨넥션 상태] - 특정 어플리케이션(프로토콜,포트)이 점유하고 있는 인터넷 대역폭(MB/s),사용량(Bytes),컨넥션 수를 검색할 수 있습니다.
    • 검색된 내용의 제목을 클릭하면, 제목별로 정렬할 수 있습니다.
  • 필터링 옵션
    • 내부 IP: 설정된 내부 IP에 대한 컨넥션 정보를 검색.
    • 원격지 IP: 설정된 원격지 IP에 대한 컨넥션 정보를 검색.
    • 프로토콜 : 설정된 IP Header의 프로토콜 번호에 대한 컨넥션 정보를 검색.
    • TCP/UDP포트 : IP Header의 프로토콜이 TCP/UDP일 경우, 설정된 포트 번호와 동일한 목적지 포트를 갖는 컨넥션 정보를 검색.
    • 컨넥션 방향 : 설정된 컨넥션 방향에 대한 컨넥션 정보를 검색.
    • 모든 TCP 연결 검색 하기 : 옵션이 꺼져 있는 경우에는 TCP 컨넥션 중, TCP연결 상태가 "ESTABLISHED"(연결됨)인 경우만 검색.
    • 컨넥션 정보 자세히 보기 :
      • 프로토콜/유지시간: 프로토콜이름(번호), 컨넥션 상태, 유지시간.
      • 시작 IP:포트(요청) : 최초에 연결을 요청한 IP및 포트번호.
      • 목적 IP:포트(요청) : 최초 연결 요청시 목적지IP 및 목적지포트번호.
      • 방향 : 최초 연결 요청시의 방향을 나타냅니다.
    • 속도/통신량 보기 옵션
      • 속도(MB/s)보기: 컨넥션의 데이타 통신 속도를 초당 Mega bytes 단위로 표시.
      • 통신량(Bytes)보기: 컨넥션의 데이타 통신량을 Byte단위로 표시.
    • 초기 컨넥션 방향 별로 검색하기: [컨넥션 방향별 상태]를 검새 시, 컨넥션 생성 시, 최초 요청 방향별로 검색.
  • 기타 용어 설명
    • 컨넥션 방향
      • -> : 단방향 , <-> : 양방향
      • LAN : 내부 네트워크
      • WAN1,WAN2: 외부 네트워크(WAN2:2 WAN제품군일 경우)
      • MGMT : 공유기 자체 네트워크
