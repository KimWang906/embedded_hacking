• 기본 분산 경로
    • 별도의 규칙으로 지정되어 있지 않은 인터넷 트래픽은 기본 분산 경로로 통신이 이루어진다.

  • 자동 인터넷 라인 백업
    • WAN1 또는 WAN2 중 임의의 WAN포트에서 장애가 발생하면 정상 동작하는 WAN 포트로 모든 통신이 자동으로 이루어 집니다.
    • 라인 백업 확인 주기

      • 설정 주기동안 각 WAN의 통신을 확인 합니다.
    • 라인 백업 확인 방법

      • 사용자가 지정한 방법들중 임의의 하나의 방법에서 연결상태가 정상으로 확인되면, 현재 라인상태는 정상으로 간주합니다.
      - ARP : 게이트웨이와 ARP 통신방법으로 연결상태를 확인합니다.
      - Ping : 게이트웨이와 Ping 통신방법으로 연결상태를 확인합니다.
      - DNS : 사용자가 지정한 도메인주소를 사용하여 각 WAN의 기본 DNS와 연결상태를 확인합니다.

  • 자동 분산 비율 설정
    • 설정된 비율대로 인터넷 트래픽을 자동 분산 시킵니다.
    • PC IP 주소 단위 분산

      • PC IP 주소를 기준으로 분산.
      [STATUS]를 클릭하면 현재 PC IP주소별 통신하는 WAN 주소를 확인 할 수 있습니다.
      단, 사용자정의 분산정책에서 'PC IP주소' 옵션값으로만 설정된 PC IP주소는 표시되지 않습니다.
      예) {192.168.0.100} - WAN1, {192.168.0.200} - WAN2
    • PC IP 주소 + 포트단위 분산

      • {PC IP 주소, 소스 포트 번호} 조합 기준으로 분산.
      즉, 한대의 PC에서 새로운 통신이 이루어 질때 마다 사용되는 WAN이 변경 될 수 있음.
      예) {192.168.0.100, 1234} - WAN1, {192.168.0.100, 4321} - WAN2
