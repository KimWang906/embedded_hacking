• 현재/전체 컨넥션 정보
    • 현재 통신하고 있는 모든 컨넥션의 갯수를 알수있습니다.
    • 전체컨넥션의 수는 라우터가 가질수있는 최대 컨넥션 수를 나타냅니다.
    • 현재 컨넥션수가 전체컨넥션 수와 같아지게 되면 더이상의 인터넷 연결을 할 수 없습니다.
  • IP별 컨넥션 정보
    • IP 주소란은 연결을 시작한 IP를 나타냅니다.
    • 이정보를 이용하여 과도한 컨넥션을 생성하는 PC를 찾아낼 수 있습니다.
    • IP 주소별 '삭제' 버튼을 클릭하시면 모든 컨넥션이 삭제됩니다. 만약 해당 IP 주소에서 파일 다운로드와 같은 통신을 하고 있다면 해당 통신은 끊어지게 됩니다
