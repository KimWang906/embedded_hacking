• 사용자 간단 설정
    • 특정 IP나 PC의 인터넷 사용을 제한하는 기능입니다.
    • 시작지 IP주소
      • 인터넷 사용을 제한 할 PC의 IP를 입력합니다.
    • 시작지 MAC 주소
      • 인터넷 사용을 제한 할 PC의 MAC을 입력합니다.
    • 허용/차단
      • 설정한 특정 IP나 PC의 인터넷 사용의 허용/차단을 선택합니다.

  • 사용자 고급 설정
      • 특정 내부PC들 및 외부PC들의 통신을 차단할 수 있으며 허용할 수도 있습니다.
    • 먼저 입력된 보안설정 규칙의 우선 순위가 가장 높습니다.
    • 방향
      • 외부->내부 : 외부에서 공유기에 연결된 내부의 PC로 통신
      • 내부->외부 : 공유기에 연결된 내부의 PC에서 외부로 통신
    • 규칙이름
      • 임의로 입력가능하며 구분이 쉽도록 입력합니다.
    • 시작지 IP 주소
      • 외부->내부 : 외부PC의 IP 주소
      • 내부->외부 : 내부PC의 IP 주소
    • 시작지 MAC주소
      • 외부->내부 : 외부PC의 MAC 주소
      • 내부->외부 : 내부PC의 MAC 주소
    • 목적지 IP 주소
      • 외부->내부 : 내부PC의 IP 주소
      • 내부->외부 : 외부PC의 IP 주소
    • 프로토콜
      • Protocol Type (any/TCP/UDP)을 선택합니다.
      • 'any'는 모든 프로토콜을 의미합니다. 'any' 선택시에는 포트번호를 입력할 수 없습니다.
    • 목적지 포트
      • 외부->내부 : 내부PC의 접근 포트 번호
      • 내부->외부 : 외부PC의 접근 포트 번호
    • 허용/차단
      • 정의한 통신 접속 형태의 허용/차단을 선택합니다.

  • 사이트 접속 차단 설정
    • 특정 IP나 PC의 특정 문자열을 포함하는 사이트 주소에 대한 접속을 제한하는 기능입니다.
    • 시작지 IP주소
      • 사이트 접속 차단할 PC의 IP를 입력합니다.
    • 시작지 MAC 주소
      • 사이트 접속 차단할 PC의 MAC을 입력합니다.
    • 차단할 사이트 입력
      • 접속을 차단하고자 하는 사이트 주소의 문자열을 입력합니다.
      • 예를들어, 'sex hangame porn' 이라고 입력하시면 다음의 사이트들이 차단 됩니다. www.sex.com / www.sex.co.kr / www.hangame.com / www.hangame.co.kr / www.hangame.com / www.porn.com

  • 사용자 스케쥴 설정하기
    • 스케쥴-요일 : 설정을 적용할 요일을 선택합니다.
    • 스케쥴-시간 : 설정을 적용할 시간대를 선택합니다.
