• WPS 설정
    • WPS 설정은 무선 설정을 초보자도 쉽게 할 수 있도록 도와 줍니다.
  • WPS 설정 방법
    • WPS 버튼(PBC)을 이용한 방법
      • 무선 공유기 또는 AP의 WPS버튼을 누릅니다.
      • 무선 LED가 꺼져있는 것을 확인합니다.
      • 무선 랜카드의 WPS버튼(PBC버튼, 무선랜카드 Utility에 가상으로 존재할 수도 있음)을 누릅니다.
      • 무선 랜카드와 무선공유기의 연결이 올바른지 확인합니다.
        • 무선 공유기의 WPS버튼을 누르는 것은 "무선 장치 추가 "버튼을 누르는 동작과 동일합니다.
        • * WPS버튼을 누른 후 2분안에 무선 랜카드의 WPS버튼을 눌러야 합니다.
    • PIN CODE입력 방법
      • 고급 옵션을 체크한 후, 무선 랜카드의 PIN CODE를 입력합니다.
      • "무선 장치 추가" 버튼을 누릅니다.
      • 랜카드 용 Utility에 있는 PIN CODE 버튼을 누릅니다.
      • 무선 랜카드와 무선공유기의 연결이 올바른지 확인합니다.
