• 내부 네트워크 정보

  • 내부 IP 주소
    • 공유기의 내부접속용 IP 주소 입니다.
    • '네트워크 관리 >> 내부 네트워크 설정' 에서 변경할 수 있습니다.
  • 서브넷마스크
    • 내부 네트워크의 논리적 그룹 범위 입니다.
  • MAC 주소
    • 공유기의 MAC 주소 입니다.
  • 동적 IP 할당 범위
    • 공유기와 연결된 PC들에게 할당할 IP 주소 범위를 나타냅니다.
    • '네트워크 관리 >> 내부 네트워크 설정' 에서 변경할 수 있습니다.
  • 동적 IP 할당수
    • 공유기가 PC들에게 할당해 준 IP 주소의 개수입니다.


  • 사용 중인 IP 주소 정보

  • 할당되어진 IP 주소 리스트를 PC의 하드웨어 주소와 함께 보여 줍니다.
